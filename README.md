# IronForge

IronForge is a React Native + Expo gym and fitness tracker demo with:

- dark high-intensity workout dashboard
- 8 training programs
- active workout tracking with timers and set progression
- workout history and PR callouts
- membership tiers and loyalty rewards
- Flex AI coach demo powered by alphinium-ai + ChatInstance

## Run locally

```bash
npm install --legacy-peer-deps
npx expo install react-dom react-native-web @expo/metro-runtime
npm run web
```
